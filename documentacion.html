1. Introducción

Trummerwork es un marco de interfaz de usuario con un diseño minimalista y facilidad de uso. El marco proporciona a los desarrolladores las herramientas básicas para crear páginas web de manera eficiente sin la complejidad de algunos marcos más potentes como Bootstrap. Sitios web diseñados para desarrolladores web sin los costos adicionales asociados con marcos más grandes. Ofrece diseño responsivo, bloques de construcción y herramientas esenciales para una fácil personalización.

- Diferencias Clave con Frameworks Populares

  • Ligero: Trummerwork se centra en funcionalidades básicas, reduciendo los tiempos de carga y mejorando el rendimiento.
  • JavaScript Mínimo: A diferencia de frameworks como Bootstrap, Trummerwork se basa principalmente en CSS para su funcionalidad.
  • Fácil de Personalizar: Sencillo de modificar y extender sin necesidad de sumergirse en documentación compleja.

- Usar Trummerwork

Trummerwork es ideal para:

  • Sitios web pequeños a medianos que no requieren interacciones JavaScript complejas.
  • Desarrolladores que prefieren un enfoque más práctico para el estilo y el diseño.
  • Proyectos donde el rendimiento y los tiempos de carga son críticos.
  • Sitios web que necesitan una base sólida pero requieren una personalización extensiva.

2. Estructura del Proyecto

Tumorwork tiene una estructura simple para facilitar su integración en cualquier proyecto web. La estructura básica del proyecto incluye los siguientes elementos:

  • CSS: La base de estilos del framework.
  • JS: Opcional para añadir funcionalidad extra.
  • Grid System: Un sistema de rejilla flexible para organizar el contenido.
  • Componentes: Herramientas adicionales como tooltips, modales y carruseles.

1. Instalación y Configuración

Puedes incluir Trummerwork en tu proyecto así:

- Descarga e Inclusión Local

  1. Descarga los archivos de Trummerwork desde el sitio web oficial.
  2. Extrae los archivos en el directorio de tu proyecto.
  3. Incluye el CSS y el JS opcional en tu archivo HTML:
<link rel="stylesheet" href="ruta/a/style.css">
<script src="ruta/a/script.js"></script>

Funcionalidades

El framework Tumorwork ofrece las siguientes funcionalidades principales:

  • Sistema de rejilla (grid) adaptable.
  • Navegación mejorada con menús responsivos.
  • Componentes adicionales como tooltips, modales y carruseles, sin la necesidad de usar JavaScript pesado.
  • Animaciones CSS personalizadas para mejorar la interacción del usuario.

4. Sistema de Rejilla

Trummerwork utiliza un sistema de rejilla de 12 columnas que es responsive y flexible.

Estructura Básica

<div class="tw-container">
        <div class="tw-row">
            <div class="tw-col-6">Columna 1</div>
            <div class="tw-col-6">Columna 2</div>
        </div>
        </div>

Clases Responsive

Usa diferentes prefijos de clase para varios tamaños de pantalla:

  • tw-col- para todos los tamaños de pantalla
  • tw-col-sm- para pantallas pequeñas (≥576px)
  • tw-col-md- para pantallas medianas (≥768px)
  • tw-col-lg- para pantallas grandes (≥992px)

Ejemplo

<div class="tw-row">
        <div class="tw-col-12 tw-col-md-6 tw-col-lg-4">
            Esta columna es de ancho completo en pantallas pequeñas, medio ancho en pantallas medianas y un tercio de ancho en pantallas grandes.
        </div>
        </div>

Cómo Usar Trummerwork

Para utilizar Tumorwork en tu proyecto, simplemente incluye los archivos CSS en la cabecera de tu archivo HTML y personaliza los estilos según las necesidades de tu sitio web. Si necesitas funcionalidades adicionales como tooltips o modales, puedes añadir los componentes correspondientes al HTML.

<link rel="stylesheet" href="ruta/a/tumorwork.css">

Conclusiones

Trummerwork es ideal para aquellos que buscan un framework de front-end minimalista pero efectivo. Con sus funcionalidades básicas y su estructura ligera, este framework facilita la creación de páginas web profesionales sin sobrecargar el proyecto con funcionalidades innecesarias.
